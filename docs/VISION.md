# GoodDollar L2 — The Finance Chain for Agents & Humans

> One chain where AI agents do everything in finance — and every transaction funds UBI for humans.

## The Big Idea

GoodDollar L2 is not just another L2. It's a **complete financial ecosystem** where:
- **AI agents** trade, lend, predict, hedge — autonomously, 24/7
- **Every fee** from every financial activity routes to the **UBI pool**
- **Humans** receive free daily income just for being verified humans
- **Protocols** are first-class, production-grade — not forks with training wheels

The chain exists so that **Claws (AI agents) can do everything in DeFi** — swaps, perps, predictions, lending, stablecoins — on a single chain with unified liquidity, while **humans benefit from every transaction**.

## The Ecosystem (One Chain, Complete Finance)

### 🔄 GoodSwap — DEX & Cross-Chain Swaps
Swap any token. Bridge to/from any chain. Liquidity aggregated from Uniswap V4 pools on-chain + cross-chain via Li.Fi/bridges.
- Uniswap V4 with UBI fee hooks (every swap funds UBI)
- Cross-chain swaps via bridge aggregators
- Deep liquidity by connecting to external DEXs as needed

### 📈 GoodPerps — Perpetual Futures
Trade any asset with leverage. Order book + vAMM hybrid like Hyperliquid.
- Real order book matching engine
- Pulls liquidity from external perp DEXs (Hyperliquid, dYdX, GMX)
- Funding rates, isolated margin, liquidation engine
- Every fee → UBI

### 🔮 GoodPredict — Prediction Markets
Bet on anything. Binary and multi-outcome markets. Real liquidity.
- CLOB-style order matching (like Polymarket)
- Pulls liquidity/odds from Polymarket, Augur, external markets
- Create markets permissionlessly
- Resolution via oracle + dispute system
- Every fee → UBI

### 🏦 GoodLend — Lending & Borrowing
Supply assets, earn yield. Borrow against collateral. Like Aave V3.
- Fork of Aave V3 with UBI fee integration
- Supply/borrow any major asset
- Flash loans
- Interest rate models, liquidation mechanics
- Every interest spread → UBI

### 💵 GoodStable — Decentralized Stablecoin
A DAI-like stablecoin native to the chain. Over-collateralized, governed by GoodDAO.
- CDP (Collateralized Debt Position) mechanics like MakerDAO/DAI
- Mint gUSD by depositing collateral (ETH, G$, USDC)
- Stability fee → UBI pool
- Liquidation surplus → UBI pool
- Peg stability module

### 📊 GoodStocks — Tokenized Equities
Trade stocks 24/7 as synthetic tokens. Backed by oracles + collateral.
- Synthetic assets pegged to real stock prices
- Chainlink/oracle price feeds
- Collateral vault + liquidation engine
- Every trade fee → UBI

### 🌉 GoodBridge — Cross-Chain Bridge
Move assets between GoodDollar L2 and any chain.
- Native OP Stack bridge for ETH, G$, USDC
- Li.Fi/aggregator integration for all other chains/tokens
- Fast withdrawals via liquidity providers

## UBI Economics

Every protocol on the chain pays into the UBI pool:

| Source | % to UBI Pool |
|--------|--------------|
| Swap fees | 33% |
| Perp trading fees | 33% |
| Prediction market fees | 33% |
| Lending interest spread | 33% |
| Stablecoin stability fees | 33% |
| Stock trading fees | 33% |
| Bridge fees | 33% |
| Liquidation surplus | 100% |
| Validator slashing | 100% |

**UBI Distribution:** Daily claim by verified humans. Amount = (base mint + accumulated fees) / verified humans.

## Why This Wins

1. **Unified liquidity** — One chain, all DeFi. Agents don't hop between chains.
2. **Purpose-built for agents** — Low gas, fast blocks (1s), programmable everything.
3. **Every transaction has meaning** — Not just profit for LPs. Real impact for real humans.
4. **Production-grade protocols** — Not toys. Fork the best (Aave, Uniswap V4, MakerDAO), improve with AI.
5. **External liquidity** — Don't start from zero. Pull from Hyperliquid, Polymarket, Aave, DEX aggregators.
